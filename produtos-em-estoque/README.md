# 📦 Controle de Produtos em Estoque

Uma aplicação simples em Java via console desenvolvida como o primeiro exercício prático de Programação Orientada a Objetos (POO). O objetivo foi compreender a criação de classes, instanciação de objetos e manipulação direta de atributos e métodos.

## 🚀 Como Testar o Projeto

Se você quiser rodar este código na sua máquina usando uma IDE (como o IntelliJ IDEA):

1. **Abra o seu projeto** na IDE.
2. **Crie um pacote (package)** chamado `OOP_01_SellProducts`.
3. **Copie os arquivos** `Product.java` e `Main.java` para dentro desse pacote.
4. **Rode a aplicação** clicando no botão de "Play" (Run) ao lado do método `main` no arquivo `Main.java`.

## 🛠️ Tecnologias e Conceitos Aplicados
* **Java**
* **Instanciação de Objetos:** Criação de múltiplos objetos (`keyboard` e `mouse`) baseados na mesma classe.
* **Manipulação de Atributos:** Atribuição direta de valores para nome, preço e quantidade em estoque.
* **Métodos com Parâmetros:** Criação da lógica de vendas (`sellProduct`) que recebe a quantidade desejada e atualiza o estoque de forma dinâmica, validando se há itens suficientes disponíveis.
